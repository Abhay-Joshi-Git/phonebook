# phonebook

To run server : 
```
cd server/
npm run dev
```

To run web app:
```
cd phonebook-web-app/
ng serve
```

Description:
```
Server is a node.js-express API server.
Web application uses Angular 8, Angular Material, Angular Flex Layout, NgRx, lodash, ts-mockito
```
